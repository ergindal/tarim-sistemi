🐙 GitHub Sync & Hosting

Veriler otomatik olarak GitHub'a kaydedilir. Uygulama GitHub Pages üzerinde çalışır — her cihazdan erişilir.

GitHub bağlantısı yapılandırılmadı
📋 Kurulum Rehberi (5 dakika)
1
GitHub'da repo oluştur
github.com → New repository → İsim: tarim-sistemi → Public → Create
2
HTML dosyasını yükle
Repo sayfasında Add file → Upload files → tarim_sistemi.html dosyasını sürükle → Commit
3
GitHub Pages aktifleştir
Repo → Settings → Pages → Branch: main → Save
Adres: https://KULLANICI.github.io/tarim-sistemi
4
Personal Access Token al
github.com → Profil → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token
Seç: repo (tüm repo izinleri) → Generate → Kopyala
5
Aşağıya bilgileri gir → Bağlan
⚠ Token cihazda şifreli saklanır, hiçbir yere gönderilmez
🔒 Token yalnızca bu cihazın localStorage'ında saklanır.
🔄 Her kayıt işleminde otomatik sync çalışır.
📜 GitHub'da her commit geçmişi görülür — eski verilere dönülebilir.
